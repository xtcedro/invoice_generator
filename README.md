# Invoice Generator

**Invoice Generator** is a desktop application built with **C++** and **GTK 4** for managing invoices, services, and customer information. It is designed to be modular, user-friendly, and easily extensible, making it suitable for small businesses or personal use.

---

## Features

- **Invoice Management**: Create, view, and edit invoices for services and product sales.
- **Customer Records**: Keep track of customer details and their transaction history.
- **Service Tracking**: Add and manage services provided to customers.
- **Cross-Platform**: Designed to run on Linux and Windows.
- **Modern GUI**: Built using GTK 4 for a sleek, responsive user interface.
- **Persistent Settings**: Configuration stored in JSON for easy customization.
- **SQLite Database**: Lightweight database integration for efficient data management.

## Requirements

### Development
- **C++17** or later
- **GTK 4**
- **Meson** and **Ninja** build systems
- **SQLite**
- **nlohmann/json** library (for configuration management)

### Runtime
- GTK 4 runtime libraries (pre-installed on most Linux systems or available for Windows).
